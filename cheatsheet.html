HTML and CSS Table Reference

Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The cell container.
<tfoot> Table Foot The set of rows defining the footer in a table.

Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns a td element should span. Place inside opening td tag.
rowspan Row Span Defines how many rows a td element should span. Place inside opening td tag.

Table Styling

Style Name Description
border Border Allows editing of the table border. Common attributes entered in this are 'size' (?px), 'style' (solid) and 'color'. No commas between attributes.
border-collapse Border Collapse Allows borders around every cell to be collapsed to create a border similar to what you would see in an Excel spreadsheet. Attribute to attach with this is 'collapse'.
width Width Sets the width of the entire table. Commonly set in pixels. 'text-align' can be utilised with this to properly display text in columns that are potentially much wider than the text itself.
